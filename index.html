Tank-Wars-DS by Gordin

A clone of the artillery game Tank Wars from 1990 for the Nintendo DS

Dependencies

You will need a Nintendo DS that is able to start homebrew or a Nintendo DS emulator.

Install

To compile you will need the latest version of devkitARM, libnds, maxmod ande maybe some other libs I forgot 0o . Linux users can use this bash-script that will download and setup everything you need to compile nds files: http://cloud.github.com/downloads/Gordin/Tank-Wars-DS/setup-libnds
